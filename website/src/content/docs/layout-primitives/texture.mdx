# Texture

Display Texture.

## Usage

<PreviewConainer>
	<Texture texture="dots">
		<div>
			<Decorator padding={1}>
				<p>Lorem Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. Duis aute irure dolor in reprehenderit in voluptate velit esse cill</p>
			</Decorator>
		</div>
	</Texture>
</PreviewConainer>

<Tabs items={['HTML', 'JSX']}>
	<Tabs.Tab>
		```html
		<div data-unitone-layout="texture -texture:dots">
			<div>
				<div data-unitone-layout="decorator -padding:1">
					<p>Lorem Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. Duis aute irure dolor in reprehenderit in voluptate velit esse cill</p>
				</div>
			</div>
		</div>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		```jsx
		import { Texture, Decorator } from '@inc2734/unitone-css';

		<Texture>
			<div>
				<Decorator padding={1}>
					<p>Lorem Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. Duis aute irure dolor in reprehenderit in voluptate velit esse cill</p>
				</Decorator>
			</div>
		</Text>
		```
	</Tabs.Tab>
</Tabs>

## Props

### <Badge>HTML</Badge> -texture:\{variation\} / <Badge>JSX</Badge> texture=\{variation\}

Set any texture.

`Dots` / `offset-dots` / `grid` / `horizontal-stripe` / `vertical-stripe` / `checker-pattern` / `graph-paper` / `slash` / `backslash` / `wave` / `solid-color`

<Tabs items={['HTML', 'JSX', 'Preview']}>
	<Tabs.Tab>
		```html
		<div data-unitone-layout="texture -texture:offset-dots">
			...
		</div>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		```jsx
		import { Texture } from '@inc2734/unitone-css';

		<Texture texture="offset-dots">
			...
		</Texture>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		<PreviewConainer>

					<Decorator padding={1}>

Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. Duis aute irure dolor in reprehenderit in voluptate velit esse cill</p>
