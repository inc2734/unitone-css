# Responsive Grid

Generates a simple grid.

## Usage

<PreviewConainer>
	<ResponsiveGrid>
		<img src="https://placehold.jp/400x300.jpg" alt="" />
		<img src="https://placehold.jp/400x300.jpg" alt="" />
		<img src="https://placehold.jp/400x300.jpg" alt="" />
	</ResponsiveGrid>
</PreviewConainer>

<Tabs items={['HTML', 'JSX']}>
	<Tabs.Tab>
		```html
		<div data-unitone-layout="responsive-grid">
			<img src="https://placehold.jp/400x300.jpg" alt="" />
			<img src="https://placehold.jp/400x300.jpg" alt="" />
			<img src="https://placehold.jp/400x300.jpg" alt="" />
		</div>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		```jsx

		</ResponsiveGrid>
		```

</Tabs>

## Props

### <Badge>HTML</Badge> -gap:\{variation\} / <Badge>JSX</Badge> gap=\{variation\}

Gap.

Can be set from `-2` to `7`. For variations `2` and above, `clamp` is used to adjust the margins so that they do not become too large on small screens. If it is `2` or more, adding `m` will shrink it to size 1 on mobile. (e.g. `2m`)

<Tabs items={['HTML', 'JSX', 'Preview']}>
	<Tabs.Tab>
		```html
		<div data-unitone-layout="responsive-grid -gap:-2">
			<img src="https://placehold.jp/400x300.jpg" alt="" />
			<img src="https://placehold.jp/400x300.jpg" alt="" />
			<img src="https://placehold.jp/400x300.jpg" alt="" />
		</div>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		```jsx
		import { ResponsiveGrid } from '@inc2734/unitone-css';

		<ResponsiveGrid gap={-2}>
			<img src="https://placehold.jp/400x300.jpg" alt="" />
			<img src="https://placehold.jp/400x300.jpg" alt="" />
			<img src="https://placehold.jp/400x300.jpg" alt="" />
		</ResponsiveGrid>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		<PreviewConainer>
