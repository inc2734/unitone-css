# Center

Center the box. Settings also allow for centering of child elements and centering of text.

## Usage

<PreviewConainer>
	<Stack>
		<Center style={{ outline: '1px dotted #6366f1' }}>Lorem ipsum dolor sit amet, consectetur adipisicing elit</Center>
		<Center style={{ outline: '1px dotted #6366f1' }}>Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam</Center>
	</Stack>
</PreviewConainer>

<Tabs items={['HTML', 'JSX']}>


### <Badge>HTML</Badge> -gutters:\{variation\} / <Badge>JSX</Badge> gutters=\{variation\}

Gutters.

Can be set from `-2` to `7`. For variations `2` and above, `clamp` is used to adjust the margins so that they do not become too large on small screens.
If it is `1` or more, adding `s` will shrink it to size `-1` on mobile. (e.g. `1s`)
If it is `2` or more, adding `m` will shrink it to size `1` on mobile. (e.g. `2m`)

<Tabs items={['HTML', 'JSX', 'Preview']}>
	<Tabs.Tab>
		```html
		<div data-unitone-layout="center -gutters:1">
			...
		</div>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		```jsx
		import { Center } from '@inc2734/unitone-css';

		<Center gutters={1}>
			...
		</Center>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		<PreviewConainer>
			<Stack>
				<Center gutters={1} style={{ outline: '1px dotted #6366f1' }}>Lorem ipsum dolor sit amet, consectetur adipisicing elit</Center>
				<Center gutters={1} style={{ outline: '1px dotted #6366f1' }}>Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam</Center>
