# Dialog

Displays modal dialog content using the native `<dialog>` element.

Open and close are controlled with `command`, `commandfor`, and `id`, without JavaScript.
`DialogContent` always outputs `closedby="any"` by default.

## Usage

<PreviewConainer>
	<Dialog>
		<DialogTrigger>
			<button
				command="show-modal"
				commandfor="dialog-preview-content"
				style={{
					backgroundColor: 'var(--unitone--color--dark-indigo)',
					color: '#fff',
					padding: 'calc(var(--unitone--s-1) / 2) var(--unitone--s-1)',
				}}
			>
				Open dialog
			</button>
		</DialogTrigger>
		<DialogContent id="dialog-preview-content">
			<Decorator backgroundColor="var(--unitone--color--background)" gap="1" padding="1s">
				<p>The content of the Dialog.</p>
				<div>
					<button
						command="request-close"
						commandfor="dialog-preview-content"
						style={{
							backgroundColor: 'var(--unitone--color--dark-indigo)',
							color: '#fff',
							padding: 'calc(var(--unitone--s-1) / 2) var(--unitone--s-1)',
						}}
					>
						Close
					</button>
				</div>
			</Decorator>
		</DialogContent>
	</Dialog>
</PreviewConainer>

## Props

### `DialogTrigger`

Use any wrapper element you want, and place a clickable element with `command="show-modal"` and `commandfor="{id}"` inside it.

### `DialogContent`

Renders a native `<dialog>` element.

- `id` is required for `commandfor`.
- `closedby="any"` is set by default.
- Place a clickable element with `command="request-close"` and `commandfor="{id}"` inside it when you want an explicit close button.

#### <Badge>HTML</Badge> --unitone--max-width / <Badge>JSX</Badge> maxWidth=\{value\}

Max width.
