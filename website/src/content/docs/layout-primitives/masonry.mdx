## Props

### <Badge>HTML</Badge> -gap:\{variation\} / <Badge>JSX</Badge> gap=\{variation\}

Gap.

Can be set from `-2` to `7`. For variations `2` and above, `clamp` is used to adjust the margins so that they do not become too large on small screens. If it is `2` or more, adding `m` will shrink it to size 1 on mobile. (e.g. `2m`)

<Tabs items={['HTML', 'JSX', 'Preview']}>
	<Tabs.Tab>
		```html
		<div data-unitone-layout="masonry -gap:-2">
			...
		</div>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		```jsx
		import { Masonry } from '@inc2734/unitone-css';

		<Masonry gap={1}>
			...
		</Masonry>
		```
	</Tabs.Tab>

### <Badge>HTML</Badge> --unitone--column-width / <Badge>JSX</Badge> columnWidth=\{value\}

Column width.

`--unitone--column-width` specifies the ideal width of the column.
