# Popover

Displays floating content anchored to the trigger using Anchor Positioning and the Popover API.

In HTML, set the native popover and anchor attributes manually.
In React, also set `popoverTarget` and `id` manually.

## Usage

<PreviewConainer>
	<Popover>
		<PopoverTrigger>
			<button popoverTarget="popover-preview-content" style={{backgroundColor: "var(--unitone--color--dark-indigo)", color: "#fff", padding: "calc(var(--unitone--s-1) / 2) var(--unitone--s-1)" }}>Open popover</button>
		</PopoverTrigger>
		<PopoverContent id="popover-preview-content">
			<PopoverDialog>The content of the Popover.</PopoverDialog>
		</PopoverContent>
	</Popover>

## Props

### `Popover`

#### <Badge>HTML</Badge> -gap:\{variation\} / <Badge>JSX</Badge> gap=\{variation\}

Controls the distance between the trigger and the content.

Can be set from `-2` to `7`. For variations `2` and above, `clamp` is used to adjust the margins so that they do not become too large on small screens.
If it is `1` or more, adding `s` will shrink it to size `-1` on mobile. (e.g. `1s`)
If it is `2` or more, adding `m` will shrink it to size `1` on mobile. (e.g. `2m`)

## Notes

- In HTML, set `id`, and `popoverTarget` manually.
- In React, use a direct child element such as `<button>` inside `PopoverTrigger`.
- This primitive depends on browser support for both the Popover API and Anchor Positioning.
