# Stack

Arranges child elements vertically at regular intervals.

## Usage

<PreviewConainer>
	<Stack>
		<p style={{ outline: '1px dotted #6366f1', padding: '1em' }}>Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.</p>
		<p style={{ outline: '1px dotted #6366f1', padding: '1em' }}>Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.</p>
		<p style={{ outline: '1px dotted #6366f1', padding: '1em' }}>Duis aute irure dolor in reprehenderit in voluptate velit esse cill</p>
	</Stack>
</PreviewConainer>

## Props

### <Badge>HTML</Badge> -gap:\{variation\} / <Badge>JSX</Badge> gap=\{variation\}

Gap.

Can be set from `-2` to `7`. For variations `2` and above, `clamp` is used to adjust the margins so that they do not become too large on small screens. If it is `2` or more, adding `m` will shrink it to size 1 on mobile. (e.g. `2m`)

<Tabs items={['HTML', 'JSX', 'Preview']}>
	<Tabs.Tab>
		```html
		<div data-unitone-layout="stack -gap:-2">
			...
		</div>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		```jsx
		import { Stack } from '@inc2734/unitone-css';

		<Stack gap={-2}>
			...
		</Stack>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		<PreviewConainer>
			<Stack gap={-2}>
				<p style={{ outline: '1px dotted #6366f1', padding: '1em' }}>Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.</p>
				<p style={{ outline: '1px dotted #6366f1', padding: '1em' }}>Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.</p>
				<p style={{ outline: '1px dotted #6366f1', padding: '1em' }}>Duis aute irure dolor in reprehenderit in voluptate velit esse cill</p>
			</Stack>
		</PreviewConainer>
	</Tabs.Tab>
</Tabs>
