# Cluster

Arranges child elements horizontally.

## Usage

<PreviewConainer>
	<Cluster>
		<div style={{ outline: '1px dotted #6366f1', padding: '1em' }}>Lorem ipsum</div>
		<div style={{ outline: '1px dotted #6366f1', padding: '1em' }}>dolor sit</div>
		<div style={{ outline: '1px dotted #6366f1', padding: '1em' }}>amet consectetur</div>
		<div style={{ outline: '1px dotted #6366f1', padding: '1em' }}>adipisicing elit</div>
		<div style={{ outline: '1px dotted #6366f1', padding: '1em' }}>sed do</div>
	</Cluster>
</PreviewConainer>

<Tabs items={['HTML', 'JSX']}>
	<Tabs.Tab>
		```html
		<div data-unitone-layout="cluster">
			<div>Lorem ipsum</div>
			<div>dolor sit</div>
			<div>amet consectetur</div>
			<div>adipisicing elit</div>
			<div>sed do</div>
		</div>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		```jsx
		import { Cluster } from '@inc2734/unitone-css';

		<Cluster>
			<div>Lorem ipsum</div>
			<div>dolor sit</div>
			<div>amet consectetur</div>
			<div>adipisicing elit</div>
			<div>sed do</div>
		</Cluster>
		```
	</Tabs.Tab>
</Tabs>

### <Badge>HTML</Badge> -gap:\{variation\} / <Badge>JSX</Badge> gap=\{variation\}

Gap.

Can be set from `-2` to `7`. For variations `2` and above, `clamp` is used to adjust the margins so that they do not become too large on small screens.
If it is `1` or more, adding `s` will shrink it to size `-1` on mobile. (e.g. `1s`)
If it is `2` or more, adding `m` will shrink it to size `1` on mobile. (e.g. `2m`)

<Tabs items={['HTML', 'JSX', 'Preview']}>
	<Tabs.Tab>
		```html
		<div data-unitone-layout="cluster -gap:-2">
			...
		</div>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		```jsx
		import { Cluster } from '@inc2734/unitone-css';

		<Cluster gap={-2}>
			...
		</Cluster>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		<PreviewConainer>
			<Cluster gap={-2}>
				<div style={{ outline: '1px dotted #6366f1', padding: '1em' }}>Lorem ipsum</div>
