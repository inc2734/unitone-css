# Container

Keep the content area within a certain width, plus appropriate margins on the left and right.

## Usage

Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. Duis aute irure dolor in reprehenderit in voluptate velit esse cill

## Props

### <Badge>HTML</Badge> -gutters:\{variation\} / <Badge>JSX</Badge> gutters=\{variation\}

Gutters.

Can be set from `-2` to `7`. For variations `2` and above, `clamp` is used to adjust the margins so that they do not become too large on small screens. If it is `2` or more, adding `m` will shrink it to size 1 on mobile. (e.g. `2m`)

<Tabs items={['HTML', 'JSX', 'Preview']}>
	<Tabs.Tab>
		```html
		<div data-unitone-layout="container -gutters:-2">
			...
		</div>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		```jsx
		import { Container } from '@inc2734/unitone-css';

		<Container gutters={-2}>
			Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. Duis aute irure dolor in reprehenderit in voluptate velit esse cill
		</Container>
		```
	</Tabs.Tab>
