# Reel

Displays child elements in a simple horizontal scroll.

## Usage

<PreviewConainer>
	<Reel>
		<img src="https://placehold.jp/400x300.jpg" alt="" />
		<img src="https://placehold.jp/400x300.jpg" alt="" />
		<img src="https://placehold.jp/400x300.jpg" alt="" />
	</Reel>
</PreviewConainer>

<Tabs items={['HTML', 'JSX']}>
	<Tabs.Tab>
		```html
		<div data-unitone-layout="reel">
			<img src="https://placehold.jp/400x300.jpg" alt="" />
			<img src="https://placehold.jp/400x300.jpg" alt="" />
			<img src="https://placehold.jp/400x300.jpg" alt="" />
		</div>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		```jsx

			<img src="https://placehold.jp/400x300.jpg" alt="" />
		</Reel>
		```
	</Tabs.Tab>

### <Badge>HTML</Badge> -gap:\{variation\} / <Badge>JSX</Badge> gap=\{variation\}

Gap.

Can be set from `-2` to `7`. For variations `2` and above, `clamp` is used to adjust the margins so that they do not become too large on small screens. If it is `2` or more, adding `m` will shrink it to size 1 on mobile. (e.g. `2m`)

<Tabs items={['HTML', 'JSX', 'Preview']}>
	<Tabs.Tab>
		```html
		<div data-unitone-layout="reel -gap:-2">
			...
		</div>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		```jsx
		import { Reel } from '@inc2734/unitone-css';

		<Reel gap={-2}>
			...
		</Reel>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		<PreviewConainer>
			<Reel gap={-2}>
				<img src="https://placehold.jp/400x300.jpg" alt="" />
				<img src="https://placehold.jp/400x300.jpg" alt="" />
				<img src="https://placehold.jp/400x300.jpg" alt="" />
			</Reel>
		</PreviewConainer>
	</Tabs.Tab>
</Tabs>
