# Both Sides

Child elements are aligned at both ends. If the width of a child element becomes too large to maintain justification, it will be placed in a single column.

## Usage

<PreviewConainer>
	<BothSides>
		<p style={{ outline: '1px dotted #6366f1', padding: '1em' }}>Lorem ipsum dolor sit amet</p>
		<ul style={{ outline: '1px dotted #6366f1', padding: '1em' }}>
			<li>Lorem</li>
			<li>ipsum</li>
			<li>dolor</li>
		</ul>
	</BothSides>
</PreviewConainer>

<Tabs items={['HTML', 'JSX']}>
	<Tabs.Tab>
		```html
		<div data-unitone-layout="both-sides">
			<p>Lorem ipsum dolor sit amet</p>
			<ul>
				<li>Lorem</li>
				<li>ipsum</li>
				<li>dolor</li>
			</ul>
		</div>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		```jsx
		import { BothSides } from '@inc2734/unitone-css';

			<p>Lorem ipsum dolor sit amet</p>
		```

</Tabs>

## Props

### <Badge>HTML</Badge> -gap:\{variation\} / <Badge>JSX</Badge> gap=\{variation\}

Gap.

Can be set from `-2` to `7`. For variations `2` and above, `clamp` is used to adjust the margins so that they do not become too large on small screens. If it is `2` or more, adding `m` will shrink it to size 1 on mobile. (e.g. `2m`)

<Tabs items={['HTML', 'JSX', 'Preview']}>
	<Tabs.Tab>
		```html
		<div data-unitone-layout="both-sides -gap:-2">
			...
		</div>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		```jsx
		import { BothSides } from '@inc2734/unitone-css';

		<BothSides gap={-2}>
			...
		</BothSides>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		<PreviewConainer>
			<BothSides gap={-2}>
				<p style={{ outline: '1px dotted #6366f1', padding: '1em' }}>Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.</p>
				<ul style={{ outline: '1px dotted #6366f1', padding: '1em' }}>
					<li>Lorem</li>
					<li>ipsum</li>
					<li>dolor</li>

</Tabs>
