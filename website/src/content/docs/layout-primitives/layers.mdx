# Layers

Elements on top of each other. Layers have a `12 x 7` (`7 x 12` in portrait mode) grid, along which child layers can be shifted.

`Layers` is not only for placing a background image behind content.
Use it when the overlap itself is part of the layout, such as:

- a hero card crossing the lower edge of the image area
- a notice panel partially outside the media block
- foreground copy and controls sharing one layered composition with the media

When that overlap boundary matters visually, keep the overlapping pieces inside the same `Layers` instance and assign explicit grid ranges to the relevant children.

Typical pattern:

```html
<div data-unitone-layout="layers -cover">
	<img src="..." alt="" style="--unitone--grid-row: 1 / -2" />

	<div
		data-unitone-layout="stack -gap:1"
		style="--unitone--grid-column: 2 / -2; --unitone--grid-row: 2 / -1"
	>
		<div>Hero copy</div>
		<div>Buttons</div>
		<div>Notice card</div>
	</div>
</div>
```

If you move the notice card outside `Layers`, the overlap becomes a spacing effect instead of a real layered composition.

## Props

### <Badge>HTML</Badge> -gap:\{variation\} / <Badge>JSX</Badge> gap=\{variation\}

Gap.

Can be set from `-3` to `7`. For variations `2` and above, `clamp` is used to adjust the margins so that they do not become too large on small screens.
If it is `1` or more, adding `s` will shrink it to size `-1` on mobile. (e.g. `1s`)
If it is `2` or more, adding `m` will shrink it to size `1` on mobile. (e.g. `2m`)

<Tabs items={['HTML', 'JSX', 'Preview']}>

		```html

			...
		</div>
		```

	<Tabs.Tab>

### Grid item position

You can set the position of child elements with the associated properties of the CSS grid.

- `align-self`: `data-unitone-layout="-align-self:end"`
- `justify-self`: `data-unitone-layout="-justify-self:end"`
- `grid-column`: `--unitone--grid-column: 1/-1`
- `grid-row`: `--unitone--grid-row: 1/-1`

<Tabs items={['HTML', 'JSX', 'Preview']}>
	<Tabs.Tab>
		```html
		<div data-unitone-layout="layers -portrait">
			<img src="..." alt="" />

			<div data-unitone-layout="-align-self:center -justify-self:center" style="--unitone--grid-column: 2/-2">
				...
			</div>
		</div>
		```
	</Tabs.Tab>
	<Tabs.Tab>

			</div>
