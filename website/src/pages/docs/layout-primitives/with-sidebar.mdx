# With Sidebar

Assemble the component with sidebars. If the configured width can no longer be maintained, a single column display will be used.

## Usage

<PreviewConainer>
	<WithSidebar>
		<p style={{ outline: '1px dotted #6366f1', padding: '1em' }}>Lorem ipsum dolor sit amet</p>
		<ul style={{ outline: '1px dotted #6366f1', padding: '1em' }}>
			<li>Lorem</li>
			<li>ipsum</li>
			<li>dolor</li>
		</ul>
	</WithSidebar>
</PreviewConainer>

<Tabs items={['HTML', 'JSX']}>
	<Tabs.Tab>
		```html
		<div data-unitone-layout="with-sidebar">
			<p>Lorem ipsum dolor sit amet</p>
			<ul>
				<li>Lorem</li>
				<li>ipsum</li>
				<li>dolor</li>
			</ul>
		</div>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		```jsx
		import { WithSidebar } from '@inc2734/unitone-css';

		<WithSidebar>
			<p>Lorem ipsum dolor sit amet</p>
			<ul>
				<li>Lorem</li>
				<li>ipsum</li>
				<li>dolor</li>
			</ul>
		</WithSidebar>
		```
	</Tabs.Tab>
</Tabs>

## Props

### Sidebar

Sets sidebar position. Default is `right`.

`left` / `right`

<Tabs items={['HTML', 'JSX', 'Preview']}>
	<Tabs.Tab>
		```html
		<div data-unitone-layout="with-sidebar -sidebar:left">
			...
		</div>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		```jsx
		import { WithSidebar } from '@inc2734/unitone-css';

		<WithSidebar sidebar="left">
			...
		</WithSidebar>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		<PreviewConainer>
			<WithSidebar sidebar="left">
				<p style={{ outline: '1px dotted #6366f1', padding: '1em' }}>Lorem ipsum dolor sit amet</p>

					<li>Lorem</li>

					<li>dolor</li>
				</ul>
			</WithSidebar>
		</PreviewConainer>
	</Tabs.Tab>
</Tabs>

### <Badge>HTML</Badge> -gap:\{variation\} / <Badge>JSX</Badge> gap=\{variation\}

Gap.

Can be set from `-2` to `7`. For variations `2` and above, `clamp` is used to adjust the margins so that they do not become too large on small screens.

<Tabs items={['HTML', 'JSX', 'Preview']}>
	<Tabs.Tab>
		```html
		<div data-unitone-layout="with-sidebar -gap:-2">
			...
		</div>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		```jsx
		import { WithSidebar } from '@inc2734/unitone-css';

			...
		</WithSidebar>
		```
	</Tabs.Tab>
