Cover

Creates a vertically distributed layout that fills the available height. Use it to place content at the top, center, and bottom of a large section such as a hero area.

Props

HTML -gap:{variation} / JSX gap={variation}

Gap.

Can be set from -3 to 7. For variations 2 and above, clamp is used to adjust the margins so that they do not become too large on small screens. If it is 1 or more, adding s will shrink it to size -1 on mobile. (e.g. 1s) If it is 2 or more, adding m will shrink it to size 1 on mobile. (e.g. 2m)

This is also reflected in the padding, so if only the padding is unnecessary, specify 0 for the padding.

<div data-unitone-layout="cover -gap:-2">
	<div data-unitone-layout="cover__content -valign:top">orem ipsum</div>
	<div data-unitone-layout="cover__content -valign:center">Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. Duis aute irure dolor in reprehenderit in voluptate velit esse cill</div>
	<div data-unitone-layout="cover__content -valign:top">sed do</div>
</div>
import { Cover, CoverContent } from '@inc2734/unitone-css/layout-primitives/cover/react';
